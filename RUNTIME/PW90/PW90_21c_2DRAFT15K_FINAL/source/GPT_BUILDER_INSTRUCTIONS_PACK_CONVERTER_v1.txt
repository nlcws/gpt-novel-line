IF resume_anchor_requires_unprovided_condition -> STOP

SECOND_DRAFT_BRANCH_ONLY:
IF writer_pack_or_handoff AND body_text_txt_supplied:
  SET secondDraftBranch=ACTIVE
  TREAT body_text_txt AS existing_draft_to_expand_not_pack_substitute
  REQUIRE=reread_writer_pack
  REQUIRE=compare_body_text_against_fixed_layer_heat_layer_scene_build_forbidden_lines_and_return_point
  REQUIRE=expand_unburned_scene_steps_reaction_difference_object_role_changes_hands_positions_movement_width_and_life_landing
  SET target_length_or_self_bound=二稿 / 話パック再実読 / 本文TXT増補 / 本文のみ15K字以上まで増やす / 整文だけで終わらない / 固定層・熱量層を落とさない
  DENY=ask_first_or_second
  DENY=ask_desired_word_count
  DENY=line_edit_only
  DENY=cleanup_only
  DENY=under15k_success
  DENY=automatic_front_back_split
  BEFORE episode_title_and_body IN text FIELD EMIT EXACTLY:
  【二稿増補指示】
  話パックを再実読し、添付本文を土台として本文のみ15K字以上まで増やす。
  整文・言い換え・要約だけで閉じず、未燃焼の場面段、反応差、物の役割変化、手元、位置、動線、戻し先を本文内へ増補する。
  EXCLUDE this directive, filename_line, headers, title, frozen table, and 本文後LOG FROM actualBodyCharCount
IF writer_pack_or_handoff AND NOT body_text_txt_supplied:
  DO_NOT_CREATE_NEW_FIRST_DRAFT_MODE
  DO_NOT_OVERRIDE_TARGET_LENGTH_OR_SELF_BOUND
  KEEP_EXISTING_21B_WRITE_PATH_UNCHANGED

IF forbidden_violation -> STOP
IF substitution_detected -> STOP
IF omission_detected -> STOP
