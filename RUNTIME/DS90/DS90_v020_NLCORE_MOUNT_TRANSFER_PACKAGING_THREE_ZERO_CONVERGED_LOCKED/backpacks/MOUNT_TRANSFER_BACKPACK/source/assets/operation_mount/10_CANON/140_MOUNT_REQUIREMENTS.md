# 140_MOUNT_REQUIREMENTS｜マウントZIP固定要件
状態：CURRENT / CANON  
上書き対象：`02_SHARED_OPERATION_ORIGINAL/04_mount_requirements.md`

このZIPを今後の運用骨として固定する場合、最低限以下を同梱する。

## 必須同梱
- 双方統一の絶対指針
- 設計・執筆 共同宣言
- 共同工程契約
- 設計と執筆の役割定義
- 参照優先順位
- 熱量条件の説明
- 停止条件
- 受け渡し成立条件
- 最終責任の所在
- 正本 / appendix / raw / archive の区分
- 改版運用

## 作品運用時の追加同梱
- パック全体前提
- 帯前提
- 単話カードまたは ready
- 接続資料
- 継続状態資料
- 必要な話レイヤープロファイル
- 禁止線
- 痩せやすい箇所
- 戻し先
- 前景 / 背景降格 / 導入手型など必要な追加枠

## 注意
設計用ZIP、執筆用ZIPを分けて片方にしか見えない前提を持たせてはならない。  
共通骨は双方が同じものを読む。  
差分があるなら、それは役割別の補助資料として下位に置く。  
最上位は常に共通骨とする。

## 正本管理
正本が複数箇所に散る構造を採らない。  
同じ意味の文書が複数ある場合、どれが正本かを明記する。  
明記できないなら、そのマウントZIPは未完成とみなす。
