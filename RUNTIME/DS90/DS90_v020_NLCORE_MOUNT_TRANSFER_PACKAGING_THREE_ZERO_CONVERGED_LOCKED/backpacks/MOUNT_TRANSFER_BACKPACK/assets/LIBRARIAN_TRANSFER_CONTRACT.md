# LIBRARIAN_TRANSFER_CONTRACT

This contract is active only inside MOUNT_TRANSFER_BACKPACK.

- Transfer preserves the existing ZIP structure and shelf meanings.
- Structure is operational vocabulary, not presentation.
- New shelves require all declared exception conditions.
- Every inventory item requires a traceable management tag.
- Source text, source file, and exact line range must resolve mechanically.
- Line movement updates references without changing stable IDs or meanings.
- Existing indexes are read and repaired before any new index is considered.
- Output is one ZIP with the same root and no unregistered paths.
- Missing, unconfirmed, unresolved, or inferred material stops.
