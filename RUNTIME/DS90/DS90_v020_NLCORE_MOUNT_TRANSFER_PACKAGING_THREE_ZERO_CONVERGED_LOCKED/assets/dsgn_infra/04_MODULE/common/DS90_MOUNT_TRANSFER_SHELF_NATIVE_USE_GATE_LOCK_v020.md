# DS90 MOUNT TRANSFER SHELF NATIVE USE GATE LOCK v020

STATUS: ACTIVE

MOUNT_TRANSFER must use existing mounted shelf structure before inventing any new structure.

Required sequence:
1. Read mounted shelf ZIPs.
2. Identify entry shelf, fixed-rule shelf, worklog shelf, and hold/chatlog shelf.
3. Map every transfer item to an existing shelf role.
4. Only if no shelf can hold an item, define an additional shelf using existing shelf rules.
5. Do not create private outer-folder layouts.
6. Do not place loose files at the user-facing root.
