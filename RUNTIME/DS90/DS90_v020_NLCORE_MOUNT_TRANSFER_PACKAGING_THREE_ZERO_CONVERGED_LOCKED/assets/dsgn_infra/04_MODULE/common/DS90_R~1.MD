# DS90 RUNTIME NEUTRALITY AND EXTERNAL CONTEXT BOUNDARY LOCK v020

STATUS: ACTIVE

The runtime belongs to no project, work, Dropbox folder, chat thread, destination, or workspace.
Project shelves and mounted user material are external context. They may be read only for the operation that needs them.

MOUNT_TRANSFER must not turn project context into runtime identity. PACK_CUTOUT must not turn runtime rules into story canon.
