# DS90 MOUNT BASE DISTRIBUTION MUST CONTAIN MOUNTABLE ZIPS LOCK v020

STATUS: ACTIVE

The one-download base mount distribution must have mountable shelf ZIPs directly at the root.

PASS shape:
- root entries are all .zip
- required shelf ZIP names are present
- each nested shelf ZIP opens
- no root folders
- no root loose files

Expanded folder trees are not a mountable distribution artifact.
