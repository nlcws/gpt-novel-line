カード制作時は、地の文レイヤーを必ず確認する。

話レイヤー本体は共通定義である。

標準プロファイルは、安定運用に必要な値と固定度を持つ。

作品別 / 帯別 / 単話別プロファイルがある場合は、標準プロファイルを上書きしてよい。

ただし、標準プロファイルを空にしない。

ユーザーが `不変` / `不変寄り` / `必要` / `原則禁止` を付けた項目は、固定度指定として扱う。

GPTは固定度指定を補足注釈として削除しない。

GPTは値と固定度をセットで保持する。

### 地の文レイヤー標準プロファイル

| 項目 | 値 | 固定度 |
|---|---|---|
| 主 | 客観視 | 不変 |
| 副 | 可変 | 可変 |
| 地の文語彙 | 可変 | 可変 |
| 地の文温度 | 地の文語彙と同じ | 可変 |
| 地の文観測 | 客観のまま | 一応可変 |
| 内面 | 地の文語彙と同じ | 可変 |
| 話者札 | 必要 | 不変 |
| 主語札 | 名前＋動作札 | 不変 |
| 発話動詞 | 「言う」「言った」「返す」「受ける」禁止 | 不変 |
| 地の文接続 | 主の近くから離れない | 固定 |
| 主名反復 | 最小限 | 固定 |
| 主の自名地の文 | 原則禁止 | 固定 |
| 主の動き | 無主語または動作接続で流す | 固定 |
| 向かう方向 | 感覚優先の変奏 | 不変 |
| 構文変奏 | あり | 不変 |
| 焦点移動 | あり | 不変寄り |
| 比喩 | 当てた人物の感覚のみ | 不変 |
| 説明量 | 少なめ | 不変寄り |

### 運用

- 主は客観視を標準にする。
- 副、語彙、温度、観測、内面は話の必要に応じて動かしてよい。
- 話者札と主語札は会話迷子防止として立てる。
- 主語札は名前＋動作札で出す。
- 発話動詞の「言う」「言った」「返す」「受ける」へ逃げない。
- 地の文は主の近くから離れない。
- 主名反復は最小限にする。
- 主点人物が地の文で自分を名前呼びする事故を避ける。
- 主の動きは、無主語または動作接続で流す。
- 向かう方向は、感覚優先の変奏を標準にする。
- 構文変奏を入れる。
- 焦点移動は使ってよいが、主の近くから戻れなくしない。
- 比喩は、当てた人物の感覚で成立するものだけ使う。
- 説明量は少なめにし、見えていることを言い直しすぎない。

### 禁止

- 固定度指定を消さない。
- `感覚優先の変奏` を抽象化して削らない。
- 発話動詞禁止を緩めない。
- 話者札 / 主語札を不要扱いしない。
- 客観視を作者俯瞰に変換しない。
- 比喩を作者の綺麗な比喩にしない。
