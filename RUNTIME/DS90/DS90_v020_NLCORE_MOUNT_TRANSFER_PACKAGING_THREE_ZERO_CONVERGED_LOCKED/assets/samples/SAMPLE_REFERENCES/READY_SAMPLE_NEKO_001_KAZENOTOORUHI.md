■パック執筆投入基準

- 8場面は最低骨。本文が痩せる場合は場面を圧縮せず、各場面内の具体物、手元、猫の位置、通れる幅、客の一拍を増やす。
- 原因説明、設定説明、便利な解決で字数を埋めない。足りない分は、店内の実務、猫の居付き方、皿や席の動きで増やす。

■この話の核
この話は「会社帰りの女ひとり客が喫茶店へ入る話」ではない。
猫の都合が先にある場所へ、人が一脚ぶん混ざっていく話。
読者が最初に受け取るべき違和感は、接客の良し悪しより先に、店の中にすでに猫の流れが出来ていることに置く。

この話で絶対に外してはいけない核は3つ。

- 鈴より少し早く、すずが客を拾うこと
- 戸口から席までの短い距離で、客より先に猫の通り道があること
- 客が受け入れられたのでなく、猫の流れの中へ一脚ぶん混ざったと読めること

■場面構成
この話は最低8場面で考える。もっと増えてよい。圧縮して4場面にすると、店の顔だけが残って猫の流れが痩せる。

- 鈴より先に拾う
- 戸口で足元を流す
- 席へ座る前に猫の配置を見る
- 水と整えで机が落ち着く
- 注文より先に匂いと皿が前へ出る
- 配膳で朔が窓ぎわへ引かれる
- 膝へすなが来て、別席にも同じ流れが回る
- 客が抜けても店はもう次の収まりへ動いている

場面1 鈴より先に拾う
場面の目的

入口で「接客された」より先に、「先に見つけられた」を置く。

必須描写
- 戸を引く前後で、すずの耳か気配が鈴より先に向く
- 木の看板の下の空気が少しぬるい
- 会社帰りの客が、まだ店に入っていないのに少しほどける

必須の順番
- 戸口の手前
- 鈴より先の反応
- 客が一拍だけ息をつける

必須会話
- 「いらっしゃいませー」

削ると痩せる所
- 鈴より先に拾うこと
- 客が入る前に少しほどけること

場面2 戸口で足元を流す
場面の目的

店の入口が人のものではなく、もう猫の通り道でもあると見せる。

必須描写
- くろめが戸口と窓ぎわのあいだの細い所を使っている
- 律がどかすのでなく流す
- 客は止められたのでなく、通れる幅だけ与えられる

必須の順番
- 足元を見る
- 黒い流れに気づく
- 律が一歩だけ入る
- 道だけ残る

必須会話
- 「くろめ」
- 「どうぞ」

削ると痩せる所
- くろめが障害物ではなく流れであること
- 律がどかさず流すこと

場面3 席へ座る前に猫の配置を見る
場面の目的

着席前に、客より先に店の席が猫で割り振られていると知らせる。

必須描写
- しろたが半席を当然の顔で使っている
- ちゃたがカウンターの端を取っている
- みいは見切れず、奥の厚みとしてある
- 核5匹以外の無名猫が見切れていてもよいが、数えない

必須の順番
- しろた
- ちゃた
- みい
- そのほかの丸みや影があれば最後

削ると痩せる所
- しろたの半席
- みいを姿でなく厚みで置くこと
- 数えないまま猫密度があること

場面4 水と整えで机が落ち着く
場面の目的

千景の整えで、店の静かな線を一本入れる。

必須描写
- コースターで一拍止まる指先
- 紙ナプキンの角が直る
- 置かれたあとに机だけが少し落ち着く
- 千景は耳しっぽで立てない

必須の順番
- 置く前に止まる
- 水が置かれる
- 紙ナプキンが直る
- 客が机の収まりを受ける

必須会話
- 「窓、寒ければ閉めますので」

削ると痩せる所
- 一拍停止
- 紙ナプキンの角
- 千景の整えが机へ落ちること

場面5 注文より先に匂いと皿が前へ出る
場面の目的

すずの猫さを、耳しっぽの反応だけでなく、気の散り方と匂いへの先行で立てる。

必須描写
- すずは客に向きながら、ちゃたや匂いや皿へ少し気を散らす
- ホットミルクとプリンを先に勧める
- 焼き菓子や甘い匂いがあるなら薄く混ぜてよい

必須の順番
- 客へ向く
- 途中で皿や猫へ気が引かれる
- それでも薦めは通る

必須会話
- 「今日はホットミルク、いいです。あとプリン、ちゃんと硬いです」
- 「あ、ちゃた、それまだ」

削ると痩せる所
- すずの気が散る一拍
- それでも接客が前へ進むこと

場面6 配膳で朔が窓ぎわへ引かれる
場面の目的

店長が客より先に窓ぎわと猫へ目をやる癖を、説明でなく順番で見せる。

必須描写
- 朔は出てきて最初に窓ぎわを見る
- しろたの背に当たる風や、席の気持ちよさに引かれる
- 言葉が少し詰まり、客席へ戻るのが半歩遅い
- 人形態のままでも群れへ紛れられる感じを薄く残してよい

必須会話
- 「そこ、気持ちよさそう……じゃなくて」
- 「すみません、少しだけ待ってください」

削ると痩せる所
- 客より先に窓ぎわを見ること
- 詰まってから客へ戻ること

場面7 膝へすなが来て、別席にも同じ流れが回る
場面の目的

客個人のやさしさで閉じず、同じ店の呼吸が別席でも同時に回ると見せる。

必須描写
- すなが膝を選ぶ
- 体温側のぬるさが一度確定する
- 二人連れや別卓が入る
- しろた / くろめ / ちゃた / すず / 千景 / 律 / 朔 の呼吸が別席でも同時に働く
- 無名猫が一匹二匹見切れてもよい

必須の順番
- 膝の重み
- すなが丸くなる
- 別の客が入る
- 同じ店の呼吸が別席へ広がる

必須会話
- 「あ、今日はそこなんですねえ」
- 「そういう日です」

削ると痩せる所
- すなが膝を選ぶこと
- 別席でも同じ店の呼吸が回ること

場面8 客が抜けても店はもう次の収まりへ動いている
場面の目的

客の感想で締めず、店の側が先に次の収まりへ移っている形で閉じる。

必須描写
- 会計で律が通れる幅を残す
- 千景が皿や札を正しい向きへ戻す
- 客が抜けても、しろた / ちゃた / くろめ / すな / みい のどこかがすでに次の場所を取っている
- ここでも数えない

必須会話
- 「ありがとうございました。また、風のある日に」
- 「なくても大丈夫です。だいたい何か通ってるので」

削ると痩せる所
- 出口までの収まり
- 客が抜けたあとの次の収まり
- 感想で締めないこと

■執筆で抜けやすい点
- 店紹介や癒やし感想へ流れて、猫の通り道が人より先にある感じが薄くなる。
- 核5匹を名鑑のように順番説明してしまい、店内の流れとして読めなくなる。
- すず / 律 / 千景 / 朔の役割差が、耳しっぽ説明やキャラ紹介へ寄る。
- 客が救われた話として閉じて、客が抜けたあとも店が次の収まりへ動く画が消える。
- みいの奥の気配、無名猫の見切れ、数えない猫密度を落とす。

■猫密度の押し方
- この回の猫密度は、`木の看板 / 戸口の鈴 / くろめの通り道 / 鈴より少し早い反応 / 足元の黒い流れ / 半席だけ貸す白い背中` を起点に上げる。
- 5匹の名前を名簿のように並べて密度を出さない。見切れる丸み、椅子下、足元、カウンター端、窓ぎわ、膝、皿の通り道で増やす。
- 圧が薄い場面では説明を足さず、猫の配置 / 重なり / 通れる幅 / 隠れて見えない部位を足す。
- 無名猫、猫又、人形態の猫又、客、他怪異が埋もれてよい。ただしこの回の核を押し流さない。
- ひとりに5、6匹まとわりついてよい。店内20匹超でもよい。総数は固定しない。
- 猫が増えた理由、正体、仕組みは説明しない。増えた結果として、席・手元・皿・通り道がどう変わるかだけを本文へ出す。
- 最後の画では、猫が片づいたのではなく、猫の都合を残したまま店が回っている状態へ戻す。
