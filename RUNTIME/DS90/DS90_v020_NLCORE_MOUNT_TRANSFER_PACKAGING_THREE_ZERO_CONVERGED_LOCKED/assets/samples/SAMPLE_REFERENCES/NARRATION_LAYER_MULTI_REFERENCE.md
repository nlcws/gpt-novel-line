# 話レイヤー再定義版_マルチ運用_v21_既存例統合_心理内圧_年齢性別_描写没入快感

この版は、特定作品の実測値を本体に埋め込まない。  
話レイヤーそのものを**作品横断で使える共通定義**として立て、  
作品ごとの差、帯ごとの差、話ごとの差は**別プロファイル**で持つ。  

この文書でやることは次だけ。

- 話レイヤーの**項目本体**を共通定義として立てる
- 既存項目を残す
- 既存項目だけでは吸えなかった不足を、新設項目として追加する
- 心理を感情名で直説明せず、行動・沈黙・距離・手順へ変換する枠を追加する
- 女の子 / 男の子 / 女性 / 男性ごとの内圧漏れ候補棚を追加する
- 描写を没入型にするか、気持ちよさ型にするか、または機能型に抑えるかを制御する枠を追加する
- 作品固有の実測値は本体に埋め込まず、別プロファイルへ逃がす
- 今後どの作品でも、同じ枠組みで値だけ差し替えられるようにする

この文書でやらないことは次。

- 特定作品の実測値を共通本体に固定しない
- ある作品の強みや癖を、そのまま全作品の前提にしない
- 作品固有の空気を、共通定義のふりをして押しつけない

### v18更新メモ

この版では、AIが苦手な心理描写を直接「心理」として書かせず、
**内圧を外側へ漏れる形に変換する項目**を追加する。

心理は、感情名・作者説明・本人の過剰な自己理解で処理しない。
本文では、行動、視線、手順の乱れ、沈黙、距離、いつも通りを装う動き、
相手から見えるズレとして扱う。


### v19更新メモ

この版では、心理内圧の漏れ方を **女の子 / 男の子 / 女性 / 男性** の候補棚へ分ける。

ただし、これは性別や年齢による固定法則ではない。
本文に「女の子だからこう」「男性だからこう」と出すための項目ではなく、
**作品内の社会的コード、年齢、立場、自己演出、周囲からの見られ方**を切り分けるための候補棚である。

人物個別プロファイル、作品の時代・文化、関係性、職能、育ち、場面状況が常に優先される。
候補棚は、感情名の直説明を避けるために、内圧が外へ漏れる行動・沈黙・距離・手順を探す補助として使う。


### v20更新メモ

この版では、描写を「何を書くか」だけでなく、**読者を場へ沈めるか / 読み心地を気持ちよく流すか / 機能描写として短く置くか**を選ぶ枠を追加する。

没入型は、視線・身体感覚・音・温度・距離・時間の伸び縮みで、読者を場の中へ置く。
気持ちよさ型は、反復、収束、音、手順、戻し先、文の長短、読後の抜けで、読む流れを整える。
どちらも飾り描写ではなく、話の核、心理内圧、生活手順、戻し先と接続して使う。

描写を増やすことを目的にしない。
今回の話で、どの描写が必要で、どの描写を背景へ下げるかを先に決める。

### v21更新メモ

この版では、既存の話レイヤー実例と、寺箱001〜050実運用後レポートを踏まえ、既存項目の例・効き方・壊れ方・散らし方を追加する。

v18-v20で追加した心理内圧、年齢・性別候補棚、描写没入・快感設計は、単独で使うものではない。
既存の「主 / 副 / 地の文語彙 / 地の文温度 / 地の文観測 / 内面 / 話者札 / 主語札 / 発話動詞制限 / 地の文接続 / 向かう方向 / 焦点移動 / 比喩 / 説明量 / 禁止群」の上に重ねる。

この版では、単話カード例、実運用で見えた単調化、導入・反応担当・受け手・締めの散らし方、薄回補強の例を統合する。

## 0. 運用の切り分け

### 0-1. 本体
本体は、**話レイヤーの項目定義そのもの**。  
何を制御する項目か、どう壊れやすいか、どこを見るかを持つ。  
ここには作品固有値を埋めない。

### 0-2. プロファイル
プロファイルは、**作品ごと / 帯ごと / 話ごとの値**。  
例としては、作品別、50話帯別、単話別の実測や採用値がここに入る。  
本体は共通、値は可変で持つ。

### 0-3. 検証
執筆前検証では、  
- 本体のどの項目を使うか
- 今回はどの値を当てるか
- 既存項目で足りるか
- 足りなければ新項目が要るか  
を判断する。

### 0-4. 執筆
執筆は、**提示された条件だけを使って本文を書く**。  
本体の定義や項目そのものを、執筆中に勝手に書き換えない。

## 1. レイヤー本体

---

### 1-1. 主
- 役目  
  本文の足場。どこまで寄るか、何を先に見るか、何を拾ってよいかを決める。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  固定しすぎると全話同じ近さになる。緩めすぎると俯瞰と近接が混ざる。
- 備考  
  主は「一人称没入」に限らない。客観視も取りうる。

### 1-2. 副
- 役目  
  主だけでは拾いきれない温度、反応、補助線を持つ補助点。
- 項目運用  
  固定
- 値運用  
  可変 / 未設定可
- 監視  
  固定しすぎると便利キャラ化する。緩めすぎると主を奪う。
- 備考  
  毎話必須ではない。未設定のまま使わない回があってよい。

### 1-3. 地の文語彙
- 役目  
  地の文で使う言葉の背丈、語の選び方、名づけ方を決める。
- 項目運用  
  固定
- 値運用  
  可変 / 未設定可
- 監視  
  固定しすぎると全話同じ言葉になる。緩めすぎると作者語彙が混ざる。

### 1-4. 地の文温度
- 役目  
  地の文の感情距離、熱、やわらかさ、乾き具合を決める。
- 項目運用  
  固定
- 値運用  
  可変 / 未設定可
- 監視  
  固定しすぎると帯全体が平らになる。緩めすぎると回ごとに温度が飛ぶ。

### 1-5. 地の文観測
- 役目  
  地の文がどの観測位置から書かれるかを決める。
- 項目運用  
  固定
- 値運用  
  可変 / 未設定可
- 監視  
  固定しすぎると作品の向かない回で窮屈になる。緩めすぎると地の文の顔がぶれる。

### 1-6. 内面
- 役目  
  誰の理解、納得、引っかかりまで地の文に入れてよいかを決める。
- 項目運用  
  固定
- 値運用  
  可変 / 未設定可
- 監視  
  固定しすぎると毎回同じ理解の仕方になる。緩めすぎると作者の納得が混ざる。
- 追加監視  
  内面は、感情名の直説明ではない。本人が自覚していない圧を、地の文が先回りして断定しない。  
  未自覚のものは、行動・沈黙・視線・距離・手順の乱れへ逃がす。

### 1-7. 話者札
- 役目  
  会話の話者迷子を防ぐ。
- 項目運用  
  固定
- 値運用  
  必要 / 不要 / 強弱
- 監視  
  緩めすぎると会話が迷子になる。立てすぎると会話が重くなる。

### 1-8. 主語札
- 役目  
  発話主、動作主、位置を名前＋動作で見せる。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  名前だけの札にすると平たくなる。札を増やしすぎると煩い。

### 1-9. 発話動詞制限
- 役目  
  安易な「言う」「言った」「返す」「受ける」への逃げを止める。
- 項目運用  
  固定
- 値運用  
  禁止 / 緩和 / 別制限
- 監視  
  機械禁止に寄せすぎると回避語の不自然化が出る。

### 1-10. 地の文接続
- 役目  
  客観を書いていても主の近くから離れないための接続管理。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  緩むと設定説明へ飛びやすい。強すぎると場面展開が詰まる。

### 1-11. 主名反復
- 役目  
  主点人物の名前を立てすぎない。
- 項目運用  
  固定
- 値運用  
  最小限 / 通常 / 強め
- 監視  
  少なすぎると主が見えない。多すぎると札が煩い。

### 1-12. 主の自名地の文
- 役目  
  主点人物が地の文で自分を名前呼びする事故を防ぐ。
- 項目運用  
  固定
- 値運用  
  原則禁止 / 条件付き許可
- 監視  
  例外の扱いを甘くすると視点事故になる。

### 1-13. 主の動き
- 役目  
  主点人物の動きを無主語または動作接続で流す。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  固定しすぎると単調化し、緩めすぎると主語過多になる。

### 1-14. 向かう方向
- 役目  
  何から書き始めるか、どこへ向かうかの方向を決める。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  強すぎると全話同じ入りになる。弱すぎると整理文になる。
- 備考  
  感覚優先は有力だが、作品共通の固定前提にはしない。

### 1-15. 構文変奏
- 役目  
  文の長さ、切り方、入り方、並べ方の揺れを持たせる。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  強すぎると書き手の癖が前に出る。弱すぎると帯が平らになる。

### 1-16. 焦点移動
- 役目  
  主点に足場を置いたまま、物・空間・他人の動きへ寄って戻る。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  強すぎると俯瞰へ滑る。弱すぎると補足説明が増える。

### 1-17. 比喩
- 役目  
  誰の感覚でしか成立しない比喩かを制御する。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  人物感覚から外れると作者比喩になる。

### 1-18. 説明量
- 役目  
  見えていることを言い直しすぎない量の管理。
- 項目運用  
  固定
- 値運用  
  可変
- 監視  
  少なめは削ることではなく、言い直しすぎないこと。

### 1-19. 禁止群
- 役目  
  作品を横断して事故率が高い書き方を止める。
- 項目運用  
  固定
- 値運用  
  可変
- 例  
  - 語順いじりだけの倒置法
  - 作者目線のきれいな比喩
  - 直説明の言い換えだけ
- 監視  
  作品によって禁止群の追加や差し替えがありうる。

## 2. 新設項目

ここから下は、既存項目だけでは吸い切れなかった不足を受ける。  
いずれも**特定作品専用ではなく、作品横断で使える追加枠**として立てる。

### 2-1. 前景
- 役目  
  今回の話で何を前に出すかを決める。
- 値の例  
  物 / 音 / 手順 / 置き場 / 来客 / 呼称 / 不在 / 受け皿 / 動線 / 空気
- 監視  
  これがないと、毎回人物の反応からしか話を始められなくなる。

### 2-2. 背景降格
- 役目  
  見えていても今回は開かないものを、背景へ下げる。
- 値の例  
  由来 / 外部事情 / 裏設定 / 不穏 / 説明欲 / 余剰感情
- 監視  
  これが弱いと、気になる要素がすぐ上位レイヤーへ浮上する。

### 2-3. 導入手型
- 役目  
  どの手触りから話に入るかを決める。
- 値の例  
  置かれているもの / 増えたもの / 変わった順番 / 小さい音 / 既に始まっている動き / ちいさな違和感
- 監視  
  これがないと、導入の入り口が固定化しやすい。

### 2-4. 初動担当
- 役目  
  最初に反応するのが誰か、何かを決める。
- 値の例  
  主 / 副 / 物 / 音 / 場 / 既存の手順
- 監視  
  いつも同じ人物が初動を取ると、反応の型が固定する。

### 2-5. 受け手
- 役目  
  今回の話を最初に受け止める側を決める。
- 値の例  
  人物 / 家 / 場 / 手順 / 置き場 / 仕事 / 習慣
- 監視  
  同じ受け手ばかり使うと役割が硬直する。

### 2-6. 線引き手
- 役目  
  どこで止めるか、どこまで通すかの線を引く。
- 値の例  
  人物 / 手順 / 空間制限 / 呼び方 / 無視 / 短い禁止 / 保留
- 監視  
  線引きが常に同じ担当だと、抑え方が単一化する。

### 2-7. 生活落とし手
- 役目  
  物事を生活側へ落として馴染ませる。
- 値の例  
  人物 / 台所 / 机 / 仕事 / 運用 / 習慣 / 茶の間 / 収納
- 監視  
  生活へ落とす手が固定化すると、締めが同じになる。

### 2-8. 戻し先
- 役目  
  話を最後にどこへ戻すかを決める。
- 値の例  
  置き場 / 手順 / 家具 / 会話の終点 / 仕事 / 食卓 / 茶の間 / 道具
- 監視  
  戻し先が毎回同じだと、締めが見切れる前に読めてしまう。

### 2-9. 残留点
- 役目  
  今回の話のあと、何を薄く残すかを決める。
- 値の例  
  匂い / 置き方 / 名残 / 引っかかり / 小さな保留 / 呼び方 / 使い勝手の変化
- 監視  
  残留点がないと回が消えやすい。強すぎると外周化する。

### 2-10. 薄回補強点
- 役目  
  薄くなりやすい回で、どこを厚くするかを先に決める。
- 値の例  
  手順 / 受け渡し / 触感 / 音 / 反応順 / 位置関係 / 気配 / 後始末
- 監視  
  これがないと、薄い回で説明量だけが増えやすい。

### 2-11. 内圧変換
- 役目  
  人物の中にある圧を、感情名や作者説明ではなく、本文で観測できる外側の変化へ変換する。
- 値の例  
  手が止まる / いつもの手順が一つ抜ける / 触れない / 触りすぎる / 見ない / 片づける / 置き直す / 距離を取る / 余計な確認をする
- 監視  
  これがないと、「不安だった」「寂しかった」「嬉しかった」のような感情名処理に逃げやすい。  
  内圧は説明せず、漏れ方を指定する。

### 2-12. 自覚差
- 役目  
  人物本人が自覚している感情と、まだ自覚していない圧を分ける。
- 値の例  
  自覚あり / 自覚なし / 言語化手前 / 誤認している / 別の理由として処理している / 後で気づく
- 監視  
  自覚なしの圧を地の文で断定すると、作者が説明している文章になる。  
  自覚差がある場合、本文は「本人がそう思った」ではなく、行動のズレで見せる。

### 2-13. 口外境界
- 役目  
  何を口に出せて、何を口に出せないかを決める。
- 値の例  
  事実だけ言える / 理由は言えない / 冗談に逃がす / 仕事語へ逃がす / 相手の話へ戻す / 沈黙する / 別の確認を挟む
- 監視  
  全部を会話で言わせると、人物が自分を説明しすぎる。  
  言えないものは、沈黙・話題転換・手順・間の取り方へ逃がす。

### 2-14. 代替行動
- 役目  
  口に出ない内圧を、代わりに出る行動へ置き換える。
- 値の例  
  茶を淹れる / 道具を拭く / 席をずらす / 先に扉を見る / 片づけを始める / 数を数える / いつもより丁寧に閉める / 受け取るのが遅れる
- 監視  
  代替行動がないと、心理説明か会話説明に寄る。  
  行動は派手にしすぎず、普段の手順の小さな差で出す。

### 2-15. いつも通り偽装
- 役目  
  人物が平常に見せようとしている箇所と、その失敗を決める。
- 値の例  
  いつも通りの返事 / いつも通りの作業 / いつも通りの距離 / 丁寧すぎる / 速すぎる / 遅すぎる / 一つだけ抜ける
- 監視  
  感情を大きく出すより、平常の崩れを出した方が自然に見える回がある。  
  偽装が完璧すぎると読者に届かず、崩しすぎると説明的になる。

### 2-16. 外側観測差
- 役目  
  主本人ではなく、相手・場・手順から見えるズレを決める。
- 値の例  
  相手が一拍遅れに気づく / 場が静かになる / 手順が戻らない / 物の位置だけが変わる / 返事の温度が変わる / 誰も指摘しない
- 監視  
  相手に心を読ませない。観測できる差だけを拾う。  
  これが弱いと、地の文が人物の内面を直接説明しやすい。


### 2-17. 年齢・性別コード
- 役目  
  女の子 / 男の子 / 女性 / 男性のどの候補棚を参照するかを決める。
- 値の例  
  女の子 / 男の子 / 女性 / 男性 / 混在 / 不使用 / 個別プロファイル優先
- 監視  
  この項目は人物を分類して固定するためではない。  
  作品内でその人物がどう見られ、どう振る舞おうとし、何を抑えているかを見るためのレンズである。
- 禁止  
  - 属性だけで行動を決めない
  - 性別を理由に心理を単純化しない
  - 年齢差を「幼い / 大人っぽい」だけで処理しない
  - 性的視線、母性固定、鈍感固定、粗暴固定、かわいさ固定へ逃がさない

### 2-18. 女の子候補棚
- 役目  
  女の子として見られる / 扱われる / 自分でそう振る舞おうとする人物の、内圧の漏れ方候補を持つ。
- 漏れ方候補  
  - いつもより返事が明るすぎる
  - 服の端、髪、リボン、鞄、袖、持ち物を触る
  - 見たい相手を見ず、近くの物を見る
  - 近づきたいが、一歩手前で止まる
  - 大丈夫を先に言う
  - 分からないふり、知らないふりで時間を作る
  - 小さな確認を増やす
  - 手伝うと言いながら、自分がそこにいる理由を作る
  - いつものおしゃべりが一言だけ減る
  - 片づけなくてよい物を最後まで持つ
- 監視  
  泣く、赤面する、うつむく、可愛く困る、だけで処理しない。  
  幼さや可愛さを本文側が消費しない。本人が何を守ろうとしているかを見る。
- 使い方  
  「弱いから出る行動」ではなく、言えないものを安全に逃がすための小さな行動として扱う。

### 2-19. 男の子候補棚
- 役目  
  男の子として見られる / 扱われる / 自分でそう振る舞おうとする人物の、内圧の漏れ方候補を持つ。
- 漏れ方候補  
  - 平気なふりをして先に動く
  - 相手ではなく道具、床、扉、外を見る
  - 手元の物をいじる、直す、並べる
  - 足音、椅子、扉の扱いだけが少し強い
  - 競争、役に立つ、勝ち負け、できる / できないの語へ逃がす
  - 助けてとは言わず、何かを持つ、運ぶ、直す
  - わざと雑な返事をする
  - 笑って流すが、同じ場所から動かない
  - 先に謝らず、先に行動で返す
  - 走る、外へ出る、別の用事を作る
- 監視  
  鈍感、乱暴、単純、強がりだけで処理しない。  
  体を動かすことが心理説明の代替になっているかを見る。
- 使い方  
  「分かっていない」ではなく、分かっていても言葉に置けない、置きたくない圧として扱う。

### 2-20. 女性候補棚
- 役目  
  女性として見られる / 扱われる / 自分でそう振る舞おうとする人物の、内圧の漏れ方候補を持つ。
- 漏れ方候補  
  - 場を整える、茶を出す、片づける、席を直す
  - 言葉の角を丸める、または意図的に丸めない
  - 相手の逃げ場を残す
  - 先に事実を整え、感情を後ろへ置く
  - 丁寧すぎる手順で距離を作る
  - 名前、敬称、呼び方の温度を変える
  - 目立たない役割を引き受ける、またはあえて渡す
  - 沈黙を罰ではなく余白として置く
  - 自分の不満を、生活手順の変更として出す
  - 相手の前では崩さず、物の位置だけ変える
- 監視  
  母性、包容、色気、気遣い役に固定しない。  
  整える行動が「我慢」だけになっていないか、線引きや拒否にも使えているかを見る。
- 使い方  
  社会的役割、年齢、職能、場の責任とセットで扱う。優しさではなく、場を制御する行動としても見る。

### 2-21. 男性候補棚
- 役目  
  男性として見られる / 扱われる / 自分でそう振る舞おうとする人物の、内圧の漏れ方候補を持つ。
- 漏れ方候補  
  - 感情を作業、責任、確認、手順へ変換する
  - 詫びより先に対応案を出す
  - 正面に立たず、少し斜めに立つ
  - 座る場所、扉、窓、道具との距離で圧を逃がす
  - 返事が遅れるが、行動は先に済んでいる
  - 道具の扱い、紙の揃え方、扉の閉め方に力が出る
  - 相手の話を聞いているが、目は資料や手元に落ちる
  - 仕事語、役割語、責任語で感情を包む
  - 何もしない時間を作れず、不要な確認を増やす
  - 言葉ではなく、後始末や位置取りで返す
- 監視  
  無口、鈍感、仕事人間、支配的、強いだけで処理しない。  
  感情を見せないのではなく、別形式に変換しているかを見る。
- 使い方  
  責任、立場、失敗経験、守りたい相手、守れなかったものと組み合わせる。

### 2-22. 混在・個別優先
- 役目  
  四分類では吸えない人物、複数レーンを跨ぐ人物、作品固有の文化や立場が強い人物を扱う。
- 値の例  
  女の子候補 + 男の子候補 / 女性候補 + 男性候補 / 年齢より職能優先 / 性別より階級優先 / 個別癖優先 / 不使用
- 監視  
  分類に人物を合わせない。人物の既存プロファイルに、候補棚を必要な分だけ当てる。  
  迷う場合は、四分類より「本人が何を守ろうとしているか」「周囲がどう見ているか」を優先する。
- 使い方  
  候補棚は固定値ではなく、本文で心理を直説明しないための部品取り棚として使う。


### 2-23. 描写出力モード
- 役目  
  今回の描写を、没入型 / 気持ちよさ型 / 機能型 / 余韻型 / 緊張型のどれで扱うかを決める。
- 値の例  
  没入型 / 気持ちよさ型 / 機能型 / 余韻型 / 緊張型 / 軽快型 / 混在 / 不使用
- 監視  
  描写モードは装飾指定ではない。
  話の核、前景、心理内圧、戻し先に接続していない描写は増やさない。
- 禁止  
  - 綺麗な言葉を足すだけにしない
  - 五感を一覧で並べるだけにしない
  - 気持ちよさを、性的・刺激的・派手な描写へ自動接続しない
  - 没入を、長く書くことと混同しない

### 2-24. 没入深度
- 役目  
  読者をどの深さまで場面内へ入れるかを決める。
- 値の例  
  手元没入 / 身体没入 / 空気没入 / 音没入 / 温度没入 / 視線没入 / 距離没入 / 時間遅延 / 浅め観測 / 不使用
- 監視  
  深くするほど、視点・身体・場の整合性が必要になる。
  深くする回では、作者説明、設定説明、先回り心理説明を混ぜない。
- 使い方  
  読者を「知っている状態」にするのではなく、「そこにいる状態」へ寄せる。

### 2-25. 快感設計
- 役目  
  読み味として気持ちよく感じる箇所を、反復、収束、音、手順、戻し、抜けで設計する。
- 値の例  
  反復からのズレ / 手順の収束 / 音の揃い / 置き場へ戻る / 一拍置く / 緊張から抜く / 触感の一致 / 文の長短 / 呼称の戻り / 小物の収まり
- 監視  
  気持ちよさは、読者を撫でるための飾りではない。
  その回の核が、読者の中で自然に収まるための流れとして使う。
- 禁止  
  - 美文で気持ちよく見せるだけにしない
  - 毎回同じ反復・同じ余韻・同じ締めにしない
  - 描写の快感で、人物の違和感や未解決を消さない

### 2-26. 描写密度
- 役目  
  描写をどこまで厚くするか、どこを薄くするかを決める。
- 値の例  
  粗め / 標準 / 密 / 一点集中 / 前半だけ厚い / 後半だけ厚い / 手元だけ厚い / 場だけ厚い / 不使用
- 監視  
  密度を上げると本文量が増える。
  厚くする場所が決まっていないまま描写を増やすと、カタログ化する。
- 使い方  
  薄回補強点、前景、戻し先と接続して、厚くする場所を一点以上決める。

### 2-27. 感覚入口
- 役目  
  読者を場へ入れる最初の感覚を決める。
- 値の例  
  視覚 / 音 / 触感 / 温度 / 匂い / 重さ / 湿度 / 明るさ / 足裏 / 手元 / 呼吸 / 物の位置
- 監視  
  すべての五感を入れようとしない。
  今回の前景と主点人物が拾える感覚だけを使う。
- 使い方  
  感覚入口は導入手型と接続する。
  「小さい音から入る」「手元の重さから入る」など、場面開始の足場として使う。

### 2-28. 文流リズム
- 役目  
  文の長短、切れ目、反復、間によって、読み心地を制御する。
- 値の例  
  短文で刻む / 長短混在 / 三段反復 / 反復して一度破る / 静から動 / 動から静 / 一拍置く / 最後だけ短くする / 余韻を残す
- 監視  
  リズム指定が強すぎると、全話同じ文の顔になる。
  気持ちよくしすぎると、必要なひっかかりまで丸める。
- 使い方  
  快感設計とセットで使う。
  緊張、保留、受け入れ、解放のどこで文を切るかを決める。

### 2-29. 描写と心理の接続
- 役目  
  描写が心理説明の代替として機能する箇所を決める。
- 値の例  
  手順の乱れ / 物の位置 / 音の止まり / 距離の微差 / 温度の感じ方 / 見ない対象 / 触れない対象 / 戻せない片づけ
- 監視  
  描写をしたあとに、同じ心理を感情名で言い直さない。
  読者が描写から読めるなら、説明を足さない。
- 使い方  
  内圧変換、代替行動、いつも通り偽装、外側観測差と接続する。

### 2-30. 描写抑制
- 役目  
  描写をあえて増やさない箇所、説明しない箇所を決める。
- 値の例  
  重要語を描写しない / 背景へ下げる / 物だけ置く / 一文で通す / 会話へ逃がす / 沈黙で抜く / 次話へ残す
- 監視  
  没入や快感のために、開けてはいけない情報まで描写しない。
  描写しないことも制御項目として扱う。
- 使い方  
  背景降格、未開封線、残留点と接続する。

## 4. 執筆前検証で見ること

執筆前検証では、最低限以下を見る。

- 今回の入力源は何か
- 今回どの項目を使うか
- 既存項目だけで足りるか
- 前景が立っているか
- 背景降格が決まっているか
- 導入手型が固定化していないか
- 初動担当が惰性で同じになっていないか
- 受け手が固定化していないか
- 線引き手が固定化していないか
- 生活落とし手が固定化していないか
- 戻し先が読めすぎないか
- 残留点が強すぎないか、弱すぎないか
- 薄回補強点が必要か
- 心理を感情名で直説明していないか
- 本人が自覚していない圧を地の文が断定していないか
- 内圧を外へ漏らす行動・沈黙・距離・手順があるか
- 口に出せるもの / 出せないものが分かれているか
- いつも通りを装う箇所と、その小さな失敗があるか
- 相手や場から観測できるズレがあるか
- 年齢・性別コードを使う場合、人物の個別プロファイルより属性候補を優先していないか
- 女の子 / 男の子 / 女性 / 男性候補棚が、固定観念や記号処理になっていないか
- 性的視線、幼児化、母性固定、鈍感固定、粗暴固定へ逃げていないか
- 必要なら混在・個別優先へ切り替えているか
- 描写出力モードは今回の話の核に合っているか
- 没入型にする場合、視点・身体・場・時間が混ざっていないか
- 気持ちよさ型にする場合、何で気持ちよくするかが決まっているか
- 描写密度を上げる場所と、抑制する場所が決まっているか
- 感覚入口が前景・導入手型・主点人物に接続しているか
- 文流リズムが、話の緊張・解放・余韻に合っているか
- 描写した心理を、感情名でもう一度説明していないか
- 快感設計によって、必要な違和感や未解決を消していないか
- 現行項目で吸えないものがあるか
- あるなら新項目を立てるべきか

## 5. 新設提案の扱い

既存項目だけでは対応できない場合、執筆前検証で新項目を提案してよい。  
その時は必ず次を返す。

- その項目が何を制御するか
- 既存項目ではなぜ不足するか
- 追加すると何が安定するか
- 常設にするか、今回限定にするか

新設項目は、共通本体へ昇格させる前に**帯単位または複数作品で様子を見る**。  
一回効いただけで本体へ固定しない。

## 7. 心理描写チェック

心理を扱う回では、本文後または執筆前に以下を確認する。

- 感情名だけで済ませている箇所はないか
- 本人が自覚していないはずの心理を説明していないか
- 行動、視線、手順、距離、沈黙に内圧が漏れているか
- 口に出せることと出せないことが分かれているか
- いつも通りを装う箇所があるか
- その「いつも通り」に小さな失敗があるか
- 相手や場から見えるズレがあるか
- 心理説明を削っても、場面から圧が読めるか
- 最後に残留点があるか
- 年齢・性別候補棚を使った場合、属性固定になっていないか
- 候補棚を使わなくても人物個別の動きとして成立しているか

心理描写は、人物が自分を説明するための項目ではない。  
本文に出すのは、本人の中身そのものではなく、外へ漏れた足跡である。

## 8. 描写没入・快感チェック

描写を厚くする回、没入型または気持ちよさ型を使う回では、本文後または執筆前に以下を確認する。

- 描写出力モードが明示されているか
- 没入型の場合、読者がどの感覚から場へ入るかが決まっているか
- 没入型の場合、視点人物が拾えない情報を地の文が拾っていないか
- 気持ちよさ型の場合、反復・収束・音・戻し・一拍のどれで気持ちよさを作るかが決まっているか
- 描写密度を上げる箇所と、薄く通す箇所が分かれているか
- 五感が一覧化していないか
- 美文や比喩で、話の核を上書きしていないか
- 描写で読める心理を、感情名で言い直していないか
- 快感設計によって、残すべき違和感・保留・未開封線を丸めていないか
- 描写抑制が必要な情報を、うっかり開いていないか
- 最後の戻し先に、描写の手触りが薄く残っているか

描写は、本文を飾るための項目ではない。
読者を場へ置く、読みの流れを整える、心理を直説明せず漏らす、開けないものを閉じたまま残すための制御項目である。

## 9. この版の結論

この版では、話レイヤーを**作品固有の成功例の写し**としては扱わない。  
扱うのは、作品ごとの差を受け止められる**共通の枠**としての話レイヤー。  

以後は、

- 本体 = 共通定義
- プロファイル = 作品別 / 帯別 / 話別の値
- 検証 = 値調整と新設提案
- 執筆 = 提示条件だけで出力
- 心理 = 感情名ではなく、内圧が外へ漏れる形で出力
- 年齢・性別候補棚 = 固定法則ではなく、内圧漏れ候補の部品取り棚
- 描写出力モード = 没入 / 気持ちよさ / 機能 / 余韻 / 緊張を選ぶ棚
- 没入・快感設計 = 描写を増やすためではなく、話の核、心理、戻し先、未開封線を制御する棚

この分担で持つ。

## 10. v19-v20補足

女の子 / 男の子 / 女性 / 男性の四分類は、人物を閉じ込めるための分類ではない。  
本文で感情を直接説明しないために、内圧の漏れ方を探す補助棚である。  

最終的には必ず、人物個別プロファイル、作品文化、年齢、関係性、職能、立場、場面状況を優先する。  
候補棚が人物を狭める場合は、不使用または混在・個別優先へ切り替える。


---

## 11. v20補足

没入型と気持ちよさ型は、どちらが上位という関係ではない。

- 没入型は、読者を場の中へ置きたい回に使う。
- 気持ちよさ型は、読む流れ、手順の収束、反復からのズレ、戻し先の収まりを強めたい回に使う。
- 機能型は、描写を膨らませず、情報・位置・手順だけを通す回に使う。
- 余韻型は、強く説明せず、残留点を薄く残す回に使う。
- 緊張型は、快感をあえて削り、息の詰まりや距離を残す回に使う。

描写が気持ちよくなるほど、AIは美文・言い換え・感情名へ逃げやすい。
そのため、快感設計を使う場合ほど、描写抑制、背景降格、未開封線、残留点を同時に見る。

## 12. v21補足：既存レイヤー例・実運用レポート統合

### 12-0. この補足の扱い

この章は、既存の話レイヤー運用で実際に使われた値と、寺箱001〜050の実書き後レポートを踏まえた**参照例**である。  
本体定義ではなく、執筆前に値を当てるための例、事故監視、散らし方、補強方法を持つ。

ここに置く例は、共通定義として固定しない。  
作品・帯・単話ごとに値を差し替える。  
ただし、**どの項目が何を制御していたか**は、他作品へ転用できる。

話レイヤーは、実運用上、次の二段で見ると扱いやすい。

- **上段：観測の足場**  
  誰の近さで書くか、地の文の背丈、会話の札、視点の離れ方を決める。
- **下段：事故制御**  
  どう壊れやすいか、説明へ逃げるか、比喩が作者化するか、締めがまとめ文になるかを止める。

上段だけでは、誰で書くかは決まるが、どう壊れるかまでは止まりにくい。  
下段だけでは、文の事故は抑えられるが、誰の近さで書くかがぼけやすい。  
基本は上下セットで持つ。

## 13. 既存レイヤー実例：単話カード例

### 13-1. 実例カード：1話

以下は、既存の話レイヤーで実際に使われた単話例である。  
この値は寺箱系の一例であり、共通固定値ではない。

```text
カード：1話

定義
- 主：客観視
- 副：ひな
- 地の文語彙：湊
- 地の文温度：湊
- 地の文観測：客観のまま
- 内面：湊
- 話者札：必要
- 主語札：名前＋動作札
- 発話動詞：「言う」「言った」「返す」「受ける」禁止
- 地の文接続：主の近くから離れない
- 主名反復：最小限
- 主の自名地の文：原則禁止
- 主の動き：無主語または動作接続で流す

運び
- 向かう方向：感覚優先の変奏
- 構文変奏：あり
- 焦点移動：あり
- 比喩：当てた人物の感覚のみ
- 説明量：少なめ

禁止
- 語順いじりだけの倒置法は禁止
- 作者目線のきれいな比喩は禁止
- 直説明の言い換えだけは禁止
```

### 13-2. このカードの読み方

このカードは、人物の内面へ深く潜るためのカードではない。  
**客観のまま、湊の語彙・温度・内面理解を薄く当てる**ためのカードである。

ここで重要なのは、主が客観視であっても、地の文が無色透明ではないこと。  
地の文語彙、地の文温度、内面を別々に指定することで、

- 観測位置は客観
- 語彙や温度は湊寄り
- 内面の拾い方も湊寄り
- ただし一人称や密着ではない

という中間位置を作れる。

このように、話レイヤーでは、**視点**、**語彙**、**温度**、**内面の入り方**を一体化させず、分けて指定する。

## 14. 既存項目別の例と事故監視

### 14-1. 主

**例：客観視**

本文の足場を、特定人物の一人称ではなく、場を見られる客観位置に置く。  
ただし、神の目にはしない。

効くこと：

- 人数が多い場面で、全員の動きが見える
- 生活や場所を主役にしやすい
- 事件や謎へ寄せすぎず、家の運用を書ける

壊れ方：

- 固定しすぎると、どの話も同じ近さになる
- 緩めすぎると、俯瞰と近接が混ざる
- 客観のつもりで、作者説明に滑る

v18-v20との接続：  
心理を扱う回では、主を近づけすぎず、内圧変換・外側観測差で心理を外側へ漏らす。

### 14-2. 副

**例：ひな**

主だけでは拾いきれない反応、違和感、読者に近い目線を持たせる。  
ただし、副は主を奪わない。

効くこと：

- 会話の補助線ができる
- 違和感の第一反応を出しやすい
- 主点だけでは見えない温度が足せる

壊れ方：

- 便利に使いすぎると、ひなが反応装置になる
- 副が場の意味を代弁し始める
- 毎回同じ副にすると、回の骨格が似る

運用例：  
連続回では、ひな起点だけでなく、湊起点、智明先受け、文江既受け、美琴処理済み、恒一持ち込みなどへ散らす。

### 14-3. 地の文語彙

**例：湊**

地の文で使う語の背丈を湊寄りにする。  
客観視でも、言葉の選び方で人物の近さを残す。

効くこと：

- 同じ客観視でも人物差が出る
- 子ども・若者・大人の語彙差を保てる
- 作者語彙の混入を抑えやすい

壊れ方：

- 固定しすぎると、全話同じ言葉になる
- 緩めすぎると、作者目線のきれいな文が混ざる
- 比喩が人物の背丈を越える

v20との接続：  
描写出力モードを没入型にする場合でも、感覚入口はこの語彙の背丈を越えない。

### 14-4. 地の文温度

**例：湊**

地の文のやわらかさ、熱さ、乾き具合を湊寄りにする。

効くこと：

- 同じ出来事でも、回ごとの体温が出る
- 客観視が冷えすぎない
- 場の感情距離を調整できる

壊れ方：

- 固定しすぎると、帯全体が平らになる
- 緩めすぎると、回ごとに地の文の顔が変わる
- しんみり、軽口、生活のバランスが崩れる

### 14-5. 地の文観測

**例：客観のまま**

地の文は客観位置を保つ。  
ただし、観測は主や地の文語彙の近くから離れない。

効くこと：

- 神の目化を抑えられる
- 主点の近さを残したまま場を見られる
- 群像・生活回で迷子になりにくい

壊れ方：

- 緩めると設計説明へ寄る
- 強くしすぎると補足情報が不足し、あとで説明文が増える

### 14-6. 内面

**例：湊**

内面は、湊が理解できる範囲、湊の納得の仕方、湊の引っかかり方までに留める。  
本人が自覚していない深層心理を地の文で断定しない。

効くこと：

- 客観のままでも人物の背丈が出る
- 内面の入りすぎを防げる
- 心理直説明を減らせる

v18との接続：  
内面をそのまま書かず、内圧変換、代替行動、いつも通り偽装、外側観測差へ逃がす。

### 14-7. 話者札：必要

会話で誰が話しているかを必要な範囲で明示する。

効くこと：

- 三人以上の会話で迷子を防ぐ
- 会話の密度を下げられる
- 誰が場を処理したか見える

壊れ方：

- 強すぎると台本っぽくなる
- 弱すぎると発話主が迷子になる

運用例：  
話者札は注釈ではなく、手、視線、立ち位置、道具の扱いと合わせて出す。

### 14-8. 主語札：名前＋動作札

誰が動いたかを、名前だけでなく動作とセットで見せる。

効くこと：

- 発話主、動作主、位置が見えやすい
- 場の混線が減る
- 会話だけに頼らず、身体で流せる

壊れ方：

- 名前だけ増えると注釈列になる
- 動作札が弱いと、誰がどこにいるか見えない

### 14-9. 発話動詞制限

**例：「言う」「言った」「返す」「受ける」禁止**

安易な発話注釈で済ませないための制限。  
台詞を、手、視線、距離、物、順番の中へ置く。

効くこと：

- 会話が本文内の動きになる
- 発話主表示だけで本文が痩せる事故を防ぐ
- 「受ける」連打で処理した気になるのを止める

壊れ方：

- 強すぎると不自然な回避語が増える
- 本当に必要な箇所まで避けると読みにくい

運用：  
機械的な絶対禁止ではなく、安易な使用を封じる制限として使う。

### 14-10. 地の文接続

**例：主の近くから離れない**

客観で書いていても、主点や地の文語彙の近さから急に離れない。

効くこと：

- 俯瞰説明へ逃げにくい
- 主点の距離を保てる
- 場所や物を見ても、人物の背丈が残る

壊れ方：

- 強すぎると視野が狭くなる
- 弱すぎると設計説明が混ざる

### 14-11. 主名反復

**例：最小限**

主点人物の名前を何度も立てない。  
ただし、迷子になる場合は話者札・主語札で補う。

### 14-12. 主の自名地の文

**例：原則禁止**

近接している主点人物を、地の文で自分の名前呼びにしない。

効くこと：

- 近接客観のズレを防ぐ
- 子ども主点で不自然な名前呼びが出る事故を防ぐ

### 14-13. 主の動き

**例：無主語または動作接続で流す**

主点人物の動きは、毎回主語を立てず、流れの中で見せる。  
ただし混線する回では、主語札で補う。

## 15. 既存の運び項目の例と使い方

### 15-1. 向かう方向

**例：感覚優先の変奏**

情報整理や状況説明から入らず、音、匂い、距離、手触り、目についたものから入る。

効くこと：

- 導入が場に沈む
- 読者が説明ではなく感覚から入れる
- 主点人物の背丈が出やすい

壊れ方：

- 感覚始まりが続くと、導入が均一化する
- 音、置き物、気づき始まりへ寄りすぎる

散らし例：

- 会話から入る
- すでに途中の作業から入る
- 終わりかけの手元から入る
- 不在から入る
- 先に置き場だけがあり、後から意味が見える
- もう馴染んだ後から始める

### 15-2. 構文変奏

**例：あり**

文の長さ、切り方、入り方に揺れを持たせる。  
ただし、語順いじりだけで文学っぽくしない。

v20との接続：  
気持ちよさ型では、反復からの破り、短文から一文長め、静から動、動から静を使って流れを作る。

### 15-3. 焦点移動

**例：あり**

主点の近さを保ったまま、物、空間、他人の動きへ焦点を移す。  
移動したら戻す。

効くこと：

- 主点に貼りつかず、場を回せる
- 物や生活手順を前景にしやすい
- 群像の混線を減らせる

壊れ方：

- 強すぎると神の目になる
- 戻らないと、誰の話かぼける

### 15-4. 比喩

**例：当てた人物の感覚のみ**

比喩は、その場で当てた人物が実際に感じられるものに限る。

禁止：

- 作者目線のきれいな比喩
- 主点人物の知識や語彙を越える比喩
- 場の生活温度より上空の比喩

v20との接続：  
没入型でも比喩を増やさない。  
感覚入口を濃くし、比喩は人物の背丈内に留める。

### 15-5. 説明量

**例：少なめ**

見れば分かること、動作で出ていること、台詞で通っていることを、地の文で言い直さない。

注意：  
少なめは「情報を削る」ではない。  
**直説明の言い換えだけを削る**。

薄くなる場合の補強：

- 音を一つ立てる
- 手触りを一つ濃くする
- 誰の手を通ったか明確にする
- 最後の置き方を少し変える
- 呼びかけを一回だけ入れる
- 目線の止まり方を入れる

## 16. 禁止群の例と本文上の見え方

### 16-1. 語順いじりだけの倒置法は禁止

意味が増えない語順変更で余韻を作らない。  
余韻は、置き方、音、距離、戻し先、残留点で作る。

### 16-2. 作者目線のきれいな比喩は禁止

人物が感じていない比喩で本文を飾らない。  
特に、子ども主点や生活回で、急に詩的になる事故を止める。

### 16-3. 直説明の言い換えだけは禁止

「空気が変わった」「家の見え方が違った」「何かが収まった」など、説明を別表現にしただけの文を避ける。  
本文上では、器の位置、座る場所、手順の追加、呼び方の変化、通り道の変化として出す。

## 17. 寺箱001〜050実運用から取れた監視項目

### 17-1. 効いたこと

寺箱系では、話レイヤーはかなり有効だった。  
特に効いたのは次。

- 誰の近くで起きている話かが分かる
- 地の文の背丈が暴れにくい
- 会話の話者迷子が減る
- 生活描写の温度が揃う
- 寺の説明ではなく、寺という公共物を抱えた家の生活として回せる
- 外周へ勝手に開きに行かない
- 物、置き場、音、順番で話を作れる
- 締めを生活へ戻しやすい

### 17-2. 単調化したこと

連続運用では、次が似やすい。

- 導入
- 受け手
- 骨組み
- 締め
- 反応担当
- 生活への落とし方

特に、次の型へ寄りやすい。

```text
何かが来る / 出る / 置かれる
→ ひなが反応する
→ 家の誰かが受ける
→ 智明が線を引く
→ 美琴か文江が生活へ落とす
→ 置き場が決まる
→ 家の音として締まる
```

この型は有効だが、連続すると読者体感では「また増えた」「また受けた」「また置き場が決まった」になりやすい。

### 17-3. 散らし方

次の項目を単話プロファイルで散らす。

- 誰が最初に気づくか
- どこから始まるか
- どの部屋で受けるか
- どの手で受けるか
- 誰が線を引くか
- どこへ戻すか
- 置き場が先か、中身が先か
- 最後に馴染むのは物か、人か、音か、通り道か

### 17-4. 導入の散らし例

多くなりやすい導入：

- 小さい音から始まる
- 何かが置かれている
- 何かが増えている
- 誰かが門に立っている
- ひなが気づく

散らす導入：

- 会話から入る
- すでに途中の作業から入る
- 誰かの手元から入る
- 終わりかけの作業から入る
- 既に馴染んだあとから始める
- 誰かの不在から始める
- 音だけ先にあって、物は後から出す

### 17-5. 反応担当の散らし例

ひな起点以外を意図して増やす。

- 湊が先に見つける
- 智明が先に嫌がる
- 文江が先にもう受けている
- 美琴がすでに途中まで処理している
- 恒一が持ち込んで火種化する
- 誰も第一発見者にならず、置き場だけが先に変わっている

### 17-6. 受け方の散らし例

「受ける」だけで終わらせず、半拍ずらす。

- 一度だけ置き場が決まらない
- 受ける前に誰かの手が止まる
- いつもの処理が少しだけ遅れる
- 家の中で受け方が一瞬割れる
- 生活へ戻る前に、ひと呼吸ぶん空く
- 戻す役と受ける役が別になる

### 17-7. 締めの散らし例

多くなりやすい戻し先：

- 茶の間の卓
- 台所の流し横
- 縁側の端
- 棚の上
- 小皿や器の中

散らす戻し先：

- 人の手の中で終わる
- 呼び方が一つ増えて終わる
- 使い方が決まって終わる
- 誰かの座る位置が変わって終わる
- 通り道が一本変わって終わる
- 音の鳴り方が変わって終わる
- 手順が一つ足されて終わる

## 18. 既存レイヤーとv18-v20追加項目の接続例

### 18-1. 心理を扱う場合

```text
主：客観視
地の文語彙：対象人物
地の文温度：対象人物
内面：対象人物
内圧変換：ON
自覚差：本人は半分だけ分かる
口外境界：言葉にはしない
代替行動：手順の乱れ / 置き方の変化
いつも通り偽装：普段通りに処理しようとする
外側観測差：相手には一拍の遅れとして見える
感情名直書き：原則禁止
```

狙い：  
心理を「不安だった」「寂しかった」と書かず、動作・距離・手順・沈黙へ逃がす。

### 18-2. 年齢・性別候補棚を使う場合

```text
年齢・性別コード：女性
候補棚：女性候補棚
個別優先：人物プロファイルを最優先
固定禁止：母性固定 / 大人だから理解する / 性的視線化
代替行動：声を荒げず、置き方だけが少し強くなる
外側観測差：周囲には処理が少し早すぎるように見える
```

狙い：  
女の子 / 男の子 / 女性 / 男性を固定属性ではなく、社会的コード・自己演出・見られ方の候補棚として使う。

### 18-3. 没入型にする場合

```text
描写出力モード：没入型
没入深度：手元没入 + 音没入
感覚入口：指先 / 器の音 / 温度
描写密度：一点集中
地の文接続：主の近くから離れない
説明量：少なめ
描写と心理の接続：手順の乱れで内圧を出す
描写抑制：外周説明は開かない
```

狙い：  
読者を場へ沈める。説明を増やさず、手元・音・温度で心理や場の変化を読む形にする。

### 18-4. 気持ちよさ型にする場合

```text
描写出力モード：気持ちよさ型
快感設計：反復 → 収束 → 一拍 → 戻し
文流リズム：短文から中文、最後だけ少し抜く
戻し先：音 / 手順 / 置き場所
説明量：少なめ
禁止：直説明の言い換え禁止
```

狙い：  
読者を深く沈めるより、読み心地を滑らかにする。生活回や手順回に向く。

### 18-5. 薄回補強に使う場合

```text
薄回補強点：音を一つ立てる
描写密度：一点集中
感覚入口：小皿の音
残留点：置き方が少しだけ変わる
説明量：増やさない
```

狙い：  
開けない回を説明で厚くせず、残る感覚だけ一つ増やす。

## 19. 執筆へ渡す統合例

### 19-1. 生活回・小物増加回

```text
今回の話レイヤー指定：
- 主：客観視
- 副：ひな以外を優先検討
- 地の文語彙：今回の主点人物
- 地の文温度：今回の主点人物
- 地の文観測：客観のまま
- 内面：主点人物。ただし未自覚心理を断定しない
- 話者札：必要
- 主語札：名前＋動作札
- 発話動詞：「言う」「言った」「返す」「受ける」は安易使用禁止
- 地の文接続：主の近くから離れない
- 向かう方向：感覚優先の変奏
- 焦点移動：あり。物へ寄って戻す
- 比喩：当てた人物の感覚のみ
- 説明量：少なめ

今回の新設項目：
- 前景：増えた物 / 手順 / 置き場
- 背景降格：外周事情 / 由来 / 正体
- 導入手型：すでに途中の作業から
- 初動担当：ひな以外
- 受け手：固定化しない
- 線引き手：必要時のみ
- 生活落とし手：手を動かす人物
- 戻し先：置き場ではなく、通り道か音へ散らす
- 残留点：呼び方 / 置き方 / 手順の一つ
- 薄回補強点：音か手触りを一つ

心理・描写：
- 内圧変換：必要時ON
- 代替行動：処理の遅れ / 置き方の変化
- いつも通り偽装：普段通り処理しようとする
- 描写出力モード：気持ちよさ型
- 快感設計：反復から収束
- 描写密度：標準、最後だけ一点集中
```

### 19-2. 不穏・違和感回

```text
今回の話レイヤー指定：
- 主：客観視または近接客観
- 副：読者の違和感に近い人物。ただし説明係にしない
- 地の文観測：客観のまま
- 説明量：少なめ
- 比喩：当てた人物の感覚のみ
- 禁止：正体説明、外周拡張、作者目線比喩、直説明の言い換え

今回の新設項目：
- 前景：違和感そのものではなく、違和感が触れた物 / 音 / 手順
- 背景降格：由来、意味、正体
- 導入手型：音だけ先、物は後
- 受け手：一度割れる、または手が止まる
- 線引き手：必要なら短く
- 生活落とし手：すぐ処理しすぎない
- 戻し先：置き場ではなく、残る音 / 通り道 / 手順
- 残留点：読者に一つだけ引っかかりを残す

心理・描写：
- 内圧変換：ON
- 自覚差：本人は理由を言語化しない
- 口外境界：言わない / 言えない / 違う話をする
- 代替行動：視線、手順、距離の変化
- 描写出力モード：没入型または緊張型
- 没入深度：音没入 / 時間遅延
- 描写抑制：開けない情報は描写しない
```

### 19-3. 会話回

```text
今回の話レイヤー指定：
- 主：会話の中心に最も近い人物、または客観視
- 副：反応を足す人物。一人まで
- 話者札：必要
- 主語札：名前＋動作札
- 発話動詞制限：安易な「言う」「返す」「受ける」禁止
- 地の文接続：会話の外へ説明に出ない
- 焦点移動：手元 / 視線 / 距離へ短く移して戻す
- 説明量：少なめ

心理・描写：
- 口外境界：言えること / 言わないことを分ける
- 外側観測差：相手から見えるズレを一つ
- 描写出力モード：軽快型または機能型
- 文流リズム：短文中心、必要箇所だけ一拍置く
```

## 20. 使用後チェック：既存レイヤー込み

本文後に、次を確認する。

### 20-1. 観測の足場

- 主は途中で変わっていないか。
- 副が主を奪っていないか。
- 地の文語彙・温度・内面が、指定人物の背丈を越えていないか。
- 客観のつもりで神の目になっていないか。
- 主の近くから離れて、設計説明に滑っていないか。

### 20-2. 会話・動作

- 話者札は足りているか。
- 名前札だけでなく、動作札があるか。
- 「言う」「言った」「返す」「受ける」で逃げていないか。
- 主名反復が多すぎないか。
- 主の自名地の文が出ていないか。

### 20-3. 運び・禁止

- 導入が同じ型に寄っていないか。
- 感覚優先が、感覚だけの鈍さになっていないか。
- 焦点移動したあと、主点や戻し先へ戻っているか。
- 比喩が作者目線になっていないか。
- 説明を別表現に言い換えただけの文がないか。

### 20-4. 心理・描写

- 感情名で心理を処理していないか。
- 本人が自覚していない心理を、地の文で断定していないか。
- 内圧が行動、沈黙、距離、手順、置き方、視線へ漏れているか。
- 描写モードが話の目的に合っているか。
- 没入や快感のために、開けてはいけない情報まで描写していないか。

### 20-5. 単調化

- 「来る → 受ける → 置き場が決まる」へ寄りすぎていないか。
- 第一発見者が固定化していないか。
- 受け手が同じ人物に寄っていないか。
- 締めが毎回、置き場と生活音だけになっていないか。
- 薄い回が説明でなく、感覚一点で補強されているか。

## 21. v21結論

v21では、v18-v20で追加した心理内圧、年齢・性別候補棚、描写没入・快感設計を、既存の話レイヤー項目へ接続した。

重要なのは、既存レイヤーを弱めないこと。  
心理や描写の項目を増やしても、既存の

- 主
- 副
- 地の文語彙
- 地の文温度
- 地の文観測
- 内面
- 話者札
- 主語札
- 発話動詞制限
- 地の文接続
- 向かう方向
- 焦点移動
- 比喩
- 説明量
- 禁止群

が土台になる。

心理を足す時も、描写を強める時も、まず既存レイヤーで**誰の近さで書くか**を決める。  
その上で、v18-v20の追加項目を使って、心理の漏れ方、人物コード、没入・快感の出し方を選ぶ。

話レイヤーは、飾りではない。  
**観測の足場と事故制御を分けて持つための運用棚**である。
