# 010_REQUIRED_READING｜必読ゲートの一本化
状態：CURRENT / CANON GATE

このファイルは、作業開始前に必ず通す読み順を一本化するためのゲートである。  
ここに書いたものを読んでいなければ、読んでいない。  
ここに書いていないものを、勝手に必読扱いへ増やさない。

---

設計、執筆、執筆側検証レーン、通し番、修正刃のどれを行う場合でも、作業開始前に次を読む。

1. `00_GATE/000_START_HERE.md`
2. `00_GATE/010_REQUIRED_READING.md`
3. `00_GATE/020_REFERENCE_PRIORITY.md`
4. `00_GATE/030_VERSION_STATUS.md`
5. `00_GATE/080_WRITING_SIDE_ACCEPTANCE_V1.md`
6. `00_GATE/081_DESIGN_SIDE_ACCEPTANCE_V1.md`
7. `00_GATE/082_JOINT_FINAL_ACCEPTANCE_V1.md`
8. `10_CANON/100_SHARED_ABSOLUTE_GUIDELINE.md`
9. `10_CANON/105_USER_DECLARATION.md`
10. `10_CANON/110_JOINT_DECLARATION.md`
11. `10_CANON/120_SHARED_PROCESS_CONTRACT.md`
12. `10_CANON/125_WRITING_SIDE_VERIFICATION_LANE.md`
13. `10_CANON/150_STANDARD_CARD_SPEC.md`

上記を読んでいない場合、入口通過扱いにしない。  
「たぶん前に読んだ」は読了ではない。  
今回の作業で使う骨として、明示的に通す。

## 2. 設計作業の追加必読

設計さんがカード、帯、接続資料、検証素材を起こす場合は、共通必読に加えて次を読む。

1. `10_CANON/151_STANDARD_CARD_TEMPLATE.md`
2. `20_ROLE_APPENDICES/220_DESIGN_DECLARATION.md`
3. `20_ROLE_APPENDICES/230_DESIGN_NECESSARY_CONDITIONS.md`
4. `20_ROLE_APPENDICES/240_CARD_TEMPLATE_UNIFICATION_REQUEST.md`
5. `20_ROLE_APPENDICES/250_LAYER_REDEFINITION_REFERENCE.md`
6. 対象プロジェクトの設計ZIP / 帯資料 / 単話資料

## 3. 執筆作業の追加必読

執筆さんが本文を書く場合は、共通必読に加えて次を読む。

1. `20_ROLE_APPENDICES/210_WRITER_DECLARATION.md`
2. `20_ROLE_APPENDICES/250_LAYER_REDEFINITION_REFERENCE.md`

執筆側検証レーンとして、カード、本文、接続、熱量、停止判断を確認する場合は、共通必読に加えて次を読む。

## 5. 共同確認・工程調整の追加必読

## 6. 改版作業の追加必読

このZIPそのものを改版する場合は、共通必読に加えて次を読む。

1. `00_GATE/040_FILE_MAP.md`
2. `00_GATE/050_MANIFEST.md`
3. `00_GATE/060_REVISION_POLICY.md`
4. 最新の `00_GATE/070_CHANGELOG_*.md`
5. 最新の `00_GATE/999_FILELIST_*.tsv`

## 7. 禁止

- 必読を読まずに「確認済み」とすること
- raw log のみ読んで canon を読んだことにすること
- archive を current より上位参照すること
- チャット断片だけで canon を上書きしたことにすること
- 執筆側検証レーンを、三者合意における第三者として扱うこと
- 片側だけの受領判定で版を上げること
- 双方最終確定宣言なしに current を変更すること
