# RUNTIME_PDF

このフォルダには、現行安定版ランタイムZIPから生成した閲覧用PDFを置く。

PDFは、各ランタイムの内容を人間・外部AI・確認作業向けに参照しやすくするための補助資料である。

## 位置づけ

- ZIP本体は [RUNTIME_ZIP/](../RUNTIME_ZIP/) を正とする。
- PDFは閲覧用の派生資料である。
- 起動・精査・移管に使用する本体はZIP側を優先する。
